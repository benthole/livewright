## Available Skills

Access shared skills via Q shortcuts:
- **QVIDEO**: Video transcript analysis and ffmpeg commands
- **QKEAP**: Keap email template creation from documents
- **QDATA**: Database queries using natural language
- **QHEATMAP**: Generate Fascinate archetype heatmaps
- **QF-CHART**: Generate primary advantage bar charts
- **QF-CHART-ANIM**: Animated bar chart generation
- **QF-PIE**: Pie chart for advantage distribution
- **QZOOM**: Zoom webinar attendance analysis
